Everything you need to know about the Aurora Collective.

What is Aurora Collective?
Aurora Collective is a decentralized ecosystem of high-end UI components and design systems built for the next generation of web applications.
How do I integrate these components?
Simply copy the HTML and CSS into your project. Each component is standalone and designed with modularity in mind, ensuring zero clashes with existing styles.
Are the components responsive?
Yes, all Aurora components are mobile-first and fully responsive, adapting seamlessly to any viewport size from mobile to ultra-wide displays.
Can I customize the aurora effects?
Absolutely. The aurora effects are managed via CSS variables, making it trivial to update colors, blur intensities, and animation speeds to match your brand.
